== Description ==
MyCompanyFiles Login generates with a shortcode ([mcf-login]) a login form for MyCompanyFiles Service. This plugin generates password recovery form too (for providers and customers).

**Features**

* Log-in form for MyCompanyFiles for customer profiles
* Log-in form for MyCompanyFiles for provider profiles
* Password recovery form for MyCompanyFiles for customer profiles
* Password recovery form for MyCompanyFiles for provider profiles

== Installation ==

**Normal installation**

1. Download the my-company-files-login.zip file to your computer.
2. Unzip the file.
3. Upload the `my-company-files-login` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Use [mcf-login] shortcode to generate the log-in form.
